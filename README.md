# Lista de la Compra

Aplicación web simple para gestionar una lista de la compra con almacenamiento en `localStorage` y gráficas mediante Chart.js.

## Uso

1. Abre `index.html` en tu navegador.
2. Añade productos indicando nombre, cantidad y, opcionalmente, categoría.
3. Marca los productos como comprados, edítalos o elimínalos según sea necesario.
4. Consulta las gráficas de productos más frecuentes y número de productos por categoría.

Todos los cambios se guardan automáticamente en tu navegador.
